# Background Info

## What is this?
This is a a project about creating a website that can create quizzes for users on different kinds of programming topics. The name of this website is Code Quiz Builder(CQB), and in order to access this website, open the main.html file in the platform-v1 folder. The scratch folder holds code and logic snippets used for testing ideas, and is not used directly for the CQB. There is also the images folder which stores the images and favicon icon that CQB uses.
